Demo: Toggle

The toggle library provides a simple target-trigger interaction whereby, when the trigger is pressed, the visibility of the target will change based on its current state.

<button data-target="ex1"><!-- trigger text --></button>

<div id="ex1" data-effect="toggle">
    <!-- toggled content -->
</div>

This is a simple content area with visibility toggled by pressing trigger with data-target="ex1". 
<button data-target="ex2"><!-- trigger text --></button>

<div id="ex2" data-effect="toggle" data-toggle="show">
    <!-- toggled content -->
</div>

This is a simple content area with visibility toggled by pressing trigger with data-target="ex1". This content area begins visible because of the toggle property data-toggle="show". 
<button data-target=".ex3"><!-- trigger text --></button>

<div class="ex3" data-effect="toggle">
    <!-- toggled content -->
</div>

<div class="ex3" data-effect="toggle">
    <!-- toggled content -->
</div>

This is a content area with the class .ex3. When trigger with data-target=".ex3" is pressed, it will toggle the visibility of this content area.
This is another content area with the class .ex3. When trigger with data-target=".ex3" is pressed, it will toggle the visibility of this content area. 
<button data-target="ex4"><!-- trigger text --></button>

<div data-effect="toggle">
    <div id="ex4">
        <!-- toggled content -->
    </div>
</div>

This is content that is a target for the trigger with data-target="ex4". However, it does not have a data-effect element. This is instead defined on a parent element. 
<button data-target="ex5"><!-- trigger text --></button>

<div id="ex5" data-effect="toggle" data-toggle="slide">
    <!-- toggled content -->
</div>

This is a simple content area with visibility toggled by pressing trigger with data-target="ex5". It also has a state of data-toggle="slide", which means that the toggle effect will slide the content rather than simply show/hide it. 
<button data-target="ex6"><!-- trigger text --></button>

    <!-- toggled content -->
</div>

This is a simple content area with visibility toggled by pressing trigger with data-target="ex5". This content area begins visible because of the toggle property data-toggle="show". It also has a state of data-toggle="slide", which means that the toggle effect will slide the content rather than simply show/hide it.
